Description

This project is a website front page that I created using HTML, CSS, and JavaScript. The goal of this project was to design a visually appealing and user-friendly homepage that could represent a real business. I focused on layout, styling, and basic interactivity to create a clean and modern design.

Technologies Used

HTML
CSS (style.css)
JavaScript

How to Run the Project
Download or clone this repository
Open the folder in VS Code
Open the index.html file
Right click and select "Open with Live Server" (or open in browser)
Features

Clean and organized layout
Styled sections using CSS
Responsive design elements
Basic interactivity using JavaScript
